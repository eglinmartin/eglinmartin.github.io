National EPC Rating Visualizer

An extension of a previous project of mine, this program used the pandas/geopandas python library to spatially aggregate EPC (Energy Performance Rating) data and display it using clipped GIS shapefiles of the United Kingdom, refined by administrative district. Such a program allows the user to easily see the spatial distribution of the UK's energy efficiency, in the hopes that this would assist policy targeting for renewable projects and retrofitting initiatives.

Belfast Urban Tree Webmap

A more detailed mapping web tool, this time using the open-source Leaflet API, that maps all urban trees in Belfast, to allow for more efficient documentation of their condition and to show areas with fewer trees. Trees are stored in an SQL database, which may be appended with more data using a form, that has been protected from SQL insertion attacks.

NOTE: Although the website is available, the SQL database is stored on the University of Leeds' servers, and therefore no information will appear on the map itself, nor will the form work as intended.

Leeds Blue Plaque Webmap

This small web map plots the distribution of blue plaques in Leeds city centre, based on their coordinates, using the Google Maps Web API. The user may zoom in closely and see detailed descriptions about the nature of the plaque and its historical context.

Wind Turbine Model

Assignment 2 of 2 for the GEOG5990M programming course at Leeds University. This program performs a simple multi-criteria analysis of the ocean floor to identify areas suitable for wind turbines.

The user may establish the parameters (minimum and maximum depth and slope), as well as both the number of turbines required and the minimum distance between them. The model runs a raster multi-criteria analysis of the ocean floor based on these parameters, clips it to only suitable land surfaces, and places a number of wind turbines in random locations. All interactivity is performed in a graphical user interface (GUI), with the program completely independent from the IDE it was developed in.

Sheep Model

Assignment 1 of 2 for the GEOG5990M programming course at Leeds University. This agent-based project models the movement of sheep through a field whilst consuming grass and avoiding wolves.

This model recreates the movement of sheep through a field. The sheep move in random directions at a constant speed through the field. Wolves patrol the field at varying speeds in random directions, and eat sheep if within a distance from them. The farmer(s) stays within the fence, shooting wolves if they come near enough to them. If sheep move within the fence, they become 'owned' by the farmer, who shears them (turning them pink), and their movement is restricted to inside the fence.
